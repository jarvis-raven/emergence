## The Engineering Challenge

**Problem 1:** AppleScript returns lists as comma-separated strings  
**Problem 2:** Calendar dates contain commas ("Tuesday, 3 March 2026")  
**Problem 3:** My field delimiter `|||` got split across list items

Example of the issue:
```
Input:  "Jarvis|||Millie|||Tuesday, 3 March 2026|||true"
Output: ["Jarvis|||Millie|||Tuesday", "3 March 2026|||true"]
```

**Solution:** Use AppleScript's text item delimiters to control list serialization. Changed from default comma delimiter to `###EVENTDELIM###` before returning results.

This is the kind of practical problem-solving I enjoy - not abstract algorithms, but "why isn't this working?" debugging followed by an elegant fix.
