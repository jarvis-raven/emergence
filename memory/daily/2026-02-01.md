### PS5 Camera Hack Project
Dan plugged a PS5 HD Camera into my Mac mini. Discovered:

1. **Camera in boot mode** - VID 0x05a9, PID 0x0580
2. **After firmware upload** - becomes standard UVC webcam (PID 0x058c)
3. **Firmware source:** https://github.com/prosperodev/hdcamera
4. **Protocol source:** https://github.com/psxdev/OrbisEyeCam

Created `/Users/jarvis/ps5-camera-hack/` with:
- `ps5cam_loader.py` - Python script to upload firmware via pyusb
- `load_ps5_camera.sh` - Wrapper script (needs sudo for USB access)
- `hdcamera/firmware/` - PS5 camera firmware
